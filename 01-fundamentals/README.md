# Chapter 01: Fundamentals

Hands-on Python basics, from simple scripts and input/output to control flow, functions, and scope.

## Status

- Completed

## Prerequisites

- Python 3.10+ (for `match/case` in `conditionals.py`)
- Terminal access (PowerShell, CMD, or Bash)

## How to run

```powershell
cd 01-fundamentals
python .\helloworld.py
```

Replace `helloworld.py` with any script in this chapter.

## Files and topics

| File | Focus |
| --- | --- |
| `helloworld.py` | Smallest Python script |
| `helloworld2.py` | Input + string concatenation |
| `helloworld3.py` | `main()` function pattern and entry point |
| `variables.py` | Core data types and comparisons |
| `variables2.py` | Lists, tuples, sets, dictionaries |
| `conditionals.py` | `if/elif/else`, ternary, `match/case` |
| `loops.py` | `while`, `for`, `break`, `continue`, `enumerate` |
| `functionbasics.py` | Parameters, defaults, return values, `*args` |
| `modules.py` | Importing and using built-in modules (`math`) |
| `variable_scope.py` | LEGB scope and `global`/`nonlocal` |
| `classes.py` | Class basics and inheritance intro |
| `palindrome.py` | Small practice challenge |
