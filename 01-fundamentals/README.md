# Chapter 1: Python Fundamentals

This chapter contains hands-on Python basics, from simple printing and input to classes.

## Prerequisites

- Python 3.10 or newer (the `match` statement in `conditionals.py` needs 3.10+)
- Terminal access (PowerShell, CMD, or Bash)

## How to run

From the project root:

```powershell
cd 01-fundamentals
python .\helloworld.py
```

Replace `helloworld.py` with any file in this chapter.

## Files and topics

| File | Focus |
| --- | --- |
| `helloworld.py` | Smallest Python script: print a message |
| `helloworld2.py` | Input + string concatenation |
| `helloworld3.py` | `main()` function pattern and `if __name__ == "__main__"` |
| `variables.py` | Core data types, reassignment, comparisons, boolean logic |
| `variables2.py` | List, tuple, set, dictionary operations |
| `conditionals.py` | `if/elif/else`, ternary expressions, logical operators, `match/case` |
| `loops.py` | `while`, `for`, `break`, `continue`, `enumerate`, nested loops, loop `else` |
| `functionbasics.py` | Parameters, defaults, return values, `*args`, functions as values |
| `modules.py` | Importing and using the built-in `math` module |
| `variable_scope.py` | LEGB scope model: local, enclosing (`nonlocal`), global, built-in |
| `classes.py` | Classes, inheritance, method overriding, and `super()` |
| `palindrome.py` | Palindrome checker function with string cleaning and reversal |
