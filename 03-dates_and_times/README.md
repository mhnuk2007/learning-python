# Chapter 03: Dates and Times

Working with `datetime` and `calendar` in Python.

## Status

- Completed

## Prerequisites

- Python 3.x

## How to run

```powershell
cd 03-dates_and_times
python .\dates.py
```

## Files and topics

| File | Focus |
| --- | --- |
| `dates.py` | Current date/time and date component access |
| `date-time-formatting.py` | `strftime()` formatting patterns |
| `time-delta.py` | Date arithmetic using `timedelta` |
| `calendars.py` | `calendar` module usage |
| `code-challenge.py` | Count weekday occurrences in a month |

## Notes
