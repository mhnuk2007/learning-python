# Learning Python

A progressive Python learning workspace organized into topic-first chapters.

## Chapters

| Chapter | Folder | Focus |
| --- | --- | --- |
| 01 | [01-fundamentals](01-fundamentals/README.md) | Python basics: syntax, variables, conditionals, loops |
| 02 | [02-file_handling](02-file_handling/README.md) | File I/O, filesystem basics, and path utilities |
| 03 | [03-dates_and_times](03-dates_and_times/README.md) | `datetime`, formatting, `timedelta`, calendar usage |
| 04 | [04-functions](04-functions/README.md) | Function definitions, parameters, return values, scope |
| 05 | [05-objects_and_classes](05-objects_and_classes/README.md) | Objects, references, mutability, and class fundamentals |
| 06 | [06-inheritance_and_overriding](06-inheritance_and_overriding/README.md) | Inheritance, subclass behavior, and method overriding |
| 07 | [07-polymorphism_encapsulation_abstraction](07-polymorphism_encapsulation_abstraction/README.md) | Core OOP principles |
| 08 | [08-modules_and_packages](08-modules_and_packages/README.md) | Imports, custom modules, and package structure |
| 09 | [09-lists_and_tuples](09-lists_and_tuples/README.md) | Sequence data structures |
| 10 | [10-stacks_and_queues](10-stacks_and_queues/README.md) | Linear data structures and operations |
| 11 | [11-sets_and_dictionaries](11-sets_and_dictionaries/README.md) | Hash-based collections |
| 12 | [12-error_handling](12-error_handling/README.md) | Exceptions and custom errors |

| 14 | [14-regular_expressions](14-regular_expressions/README.md) | Pattern matching with `re` |
| 15 | [15-file_operations_advanced](15-file_operations_advanced/README.md) | Advanced file workflows and serialization |
| 16 | [16-misc_tools](16-misc_tools/README.md) | Useful stdlib tools (`random`, `itertools`, `functools`) |

## Quick start

```powershell
# Fundamentals
cd 01-fundamentals
python .\helloworld.py
```

```powershell
# OOP principles
cd 07-polymorphism_encapsulation_abstraction
python .\polymorphism.py
```

```powershell
# Error handling
cd 12-error_handling
python .\exception-handling.py
```

```powershell
# Modules and packages
cd 08-modules_and_packages
python .\main.py
```

## Prerequisites

- Python 3.10+
- Terminal access (PowerShell, CMD, or Bash)
