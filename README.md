## Quick start

Pick a chapter and run a script:

```powershell
# Fundamentals - Hello World
cd 01-fundamentals && python .\helloworld.py

# Variables and data types
cd 01-fundamentals && python .\variables.py

# File handling
cd 02-file_handling && python .\filehandling1.py

# Dates and times
cd 03-dates_and_times && python .\dates.py

# Functions
cd 04-functions && python .\basic-function.py

# Objects and classes
cd 05-objects_and_classes && python .\objects.py

# Inheritance
cd 06-inheritance_and_overriding && python .\inheritance1.py

# OOP principles (Polymorphism, Encapsulation, Abstraction)
cd 07-polymorphism_encapsulation_abstraction && python .\polymorphism.py

# Modules and packages
cd 08-modules_and_packages && python .\main.py

# Lists and tuples
cd 09-lists_and_tuples && python .\lists.py

# Stacks and queues
cd 10-stacks_and_queues && python .\stacks.py

# Sets and dictionaries
cd 11-sets_and_dictionaries && python .\sets_and_dictionaries.py

# Conditional execution
cd 12-conditional_execution && python .\conditionals.py

# Loops
cd 13-loops && python .\for-loops.py

# Error handling
cd 14-error_handling && python .\custom-errors.py

# Comprehensions
cd 15-comprehensions && python .\comprehensions.py

# Regular expressions
cd 16-regular_expressions && python .\regex-basics.py

# Advanced file operations
cd 17-file_operations_advanced && python .\json-csv-demo.py

# Miscellaneous tools
cd 18-misc_tools && python .\misc-tools-demo.py

# Polling and event-driven
cd 19-polling_and_event_driven_programming && python .\polling-sensor.py

# Collections (Lists, Dictionaries, Tuples)
cd 20-collections && python .\list.py

# Iteration techniques
cd 21-iteration && python .\list-iteration.py

# Strings
cd 22-strings && python .\strings.py

# Input/Output
cd 23-input_output && python .\io-process.py

# Debugging
cd 24-introduction_to_debugging && python .\debugging_demo.py
```

## Prerequisites

- Python 3.10+
- Terminal access (PowerShell, CMD, or Bash)
