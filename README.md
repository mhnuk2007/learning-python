# Learning Python

This repository contains chapter-based Python practice code covering Python fundamentals, file handling, and date/time operations.

## Chapters

| Chapter | Topic | Description |
| --- | --- | --- |
| [Chapter 1](chapter1-Python-Basics/README.md) | Python Fundamentals | Basics, variables, functions, classes, loops, exception handling |
| [Chapter 2](chapter2-File-Handling/README.md) | File Handling | File I/O, path utilities, filesystem operations |
| [Chapter 3](chapter3-Dates-Times/README.md) | Dates & Times | Date/time manipulation, formatting, calendars |

## Quick start

```powershell
# Chapter 1 - Python Basics
cd chapter1-Python-Basics
python .\helloworld.py
```

```powershell
# Chapter 2 - File Handling
cd chapter2-File-Handling
python .\filehandling2.py
```

```powershell
# Chapter 3 - Dates & Times
cd chapter3-Dates-Times
python .\dates.py
```

## Prerequisites

- Python 3.10 or newer (for `match/case` statements in Chapter 1)
- Terminal access (PowerShell, CMD, or Bash)
